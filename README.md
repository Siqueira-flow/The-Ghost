# The-Ghost
Aqui está o primeiro projeto que fiz na godot engine, que nomeei de The Ghost. Tudo foi feito por Siqueira, com excessão das músicas e fontes.

Neste meu projeto o jogador é um fantasma, que tem que chegar ao seu corpo, e durante as fases tem que se esconder dos inimigos.

CONTROLES

Movimentação : AWSD
Se esconder : F
Sair do jogo : ESC

Se puder testar esse "jogo", e dar a sua opinião no meu canal do youtube, sobre o que posso melhorar nessa minha jornada de Dev, eu ficaria agradecido.
